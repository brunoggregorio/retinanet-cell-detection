def mynet.utils.colors.label_color (   label)

Return a color from a set of predefined colors.

Contains 80 colors in total.

Parameters
labelThe label to get the color for.
Returns
A list of three values representing a RGB color.

If no color is defined for a certain label, the color green is returned and a warning is printed.
